# Algoritmos y Estructuras de Datos

Entrega final de Algoritmos y Estructuras de Datos, realizada por un equipo de dos estudiantes para la carrera de Analista en Tecnologías de la Información de la Universidad ORT, desarrollada con Java 8.
Consiste en un sistema ficticio de gestión de bicicletas públicas que administra estaciones, bicicletas y alquileres. La clase Sistema oficia como base de datos ficticia; en este curso no se abarca una implementación real.

## Enfoque técnico:

- Implementación de TADs: Listas enlazadas, Pilas y Colas sin colecciones nativas de Java
- Arquitectura en capas: Dominio, TADs, Interfaces y Sistema con bajo acoplamiento
- Programación contra interfaces: Sistema desacoplado de la implementación interna de los TADs
- Algoritmos recursivos: Operaciones específicas implementadas con recursividad obligatoria
- Estructuras de datos optimizadas: Selección justificada según eficiencia temporal y espacial

## Funcionalidades:

- CRUD de estaciones, usuarios y bicicletas (URBANA, MOUNTAIN, ELECTRICA)
- Sistema de alquileres con asignación automática desde colas de espera
- Gestión de mantenimiento y depósito
- Reportes estadísticos: ocupación por barrio, ranking de uso, análisis matricial de estaciones

Este proyecto fue desarrollado siguiendo estrictamente las buenas prácticas y estándares vistos en el curso, prohibiendo el uso de colecciones nativas de Java para profundizar en el entendimiento de estructuras de datos fundamentales.
